# ToneCurve Color Generator

**Generate color palettes • HSL & OKLCH color spaces • Tailwind CSS ready • Figma variables & styles**

## Description

ToneCurve Color Generator is a powerful Figma plugin that helps designers create stunning color palettes and scales. Whether you're building a design system, working with Tailwind CSS, or need consistent brand colors, this plugin makes color generation simple and intuitive.

Create palettes using HSL sliders or an interactive gradient editor, then export them as Figma frames, styles, or variables. Perfect for design systems, brand guidelines, and Tailwind CSS projects.

## How to Use

### Quick Start
1. Open the plugin in Figma
2. Choose **HSL** (sliders), **TAILWIND** (presets), or **OKLCH** (advanced)
3. Create your color palette with live preview
4. Export as frames, styles, or variables

### HSL Mode
1. Adjust the **Hue**, **Saturation**, and **Lightness** sliders
2. Set start and end points for your color range
3. Choose an easing function or create custom Bézier curves
4. Select your color steps (Tailwind, Material, or custom)

### TAILWIND Mode
1. Use the **Lightness Range** and **Hue Shift** controls
2. Adjust **Contrast** for optimal color distribution
3. Fine-tune with **Saturation Curve** presets
4. Perfect for generating Tailwind-compatible color scales

### OKLCH Mode
1. Control **Lightness**, **Chroma**, and **Hue** independently
2. Benefit from perceptually uniform color space
3. Achieve more consistent visual results
4. Ideal for professional color systems

### Presets
1. Open the Presets section and browse groups (Tailwind CSS, Material Design, Apple Design, Brand Colors)
2. Click a preset to load its full color scale and base color
3. The step count adapts to the preset; adjust or rename as needed
4. Continue fine‑tuning with any mode or the swatch edit popup

### Compare Palettes
1. Use “Copy palette” on a compare card to duplicate the current palette
2. Click any card to make it active; its controls and edits apply to the active card
3. Rename a card by editing its title; delete extra cards when finished
4. Edit individual swatches via the edit popup (supports custom swatch names); lock or auto‑adjust as needed
5. Copied palettes render immediately; single‑palette mode expands to fill the panel

### Advanced Features
- **Color History**: Recent colors are automatically saved in the sidebar
- **Saved Palettes**: Save current palettes for future use
- **Color Extraction**: Click the eyedropper to pick colors from selected elements
- **Manual Adjustments**: Fine-tune individual colors with HSL delta controls
- **Undo/Redo**: Use Ctrl+Z/Ctrl+Y or the toolbar buttons

### Export Options
- **Frames**: Visual swatches with customizable information display
  - Show/hide HSL and RGB values
  - Display contrast ratios against white/black
  - Auto-update existing frames for workflow efficiency
- **Styles**: Reusable Figma color styles with organized naming
  - Automatic style creation and updates
  - Hierarchical naming (e.g., "Blue/500", "Blue/600")
  - Include color descriptions with hex and HSL values
- **Variables**: Design system variables for team collaboration
  - Integration with existing variable collections
  - Smart grouping and organization
  - Bulk variable creation and updates

## Installation

1. Download the plugin files
2. In Figma: `Plugins` → `Development` → `Import plugin from manifest...`
3. Select the `manifest.json` file
4. Start creating beautiful color palettes!

## Workflow Tips

### Keyboard Shortcuts
- **Ctrl+Z / Ctrl+Y**: Undo/Redo changes
- **Click + Drag**: Adjust sliders and color handles
- **Shift + Click**: Fine adjustment mode for precise control

### Pro Tips
1. **Use Color History**: Recently used colors appear in the sidebar for quick access
2. **Save Frequent Palettes**: Keep your go-to color schemes saved for reuse
3. **Extract from Design**: Use the eyedropper tool to pull colors from existing elements
4. **Advanced Curves**: Create custom easing with the Bézier editor for unique color transitions
5. **Batch Adjustments**: Use manual mode to apply HSL deltas across all colors simultaneously
6. **Live Updates**: Enable frame auto-update to see changes in real-time

### Workflow Integration
- **Variables First**: Create variables for design tokens that sync across your team
- **Styles for Components**: Use styles for reusable color applications
- **Frames for Documentation**: Generate visual documentation with contrast info
